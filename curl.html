Linux Command: curl

Transfer data from or to a server using various protocols

Command Overview

The curl command is a versatile tool for transferring data with URLs. It supports numerous protocols including HTTP, HTTPS, FTP, FTPS, SCP, SFTP, TFTP, DICT, TELNET, LDAP, and FILE.

Basic Syntax

curl [options] URL         # Basic usage
curl -o file URL          # Save output to file
curl -O URL              # Save with remote filename
curl -X METHOD URL       # Specify HTTP method

Common Options

  • -o, --output - Write to file
  • -O, --remote-name - Save using remote filename
  • -I, --head - Show document info only
  • -L, --location - Follow redirects
  • -X, --request - Specify request method

Common Examples

# Download a file
curl -O https://example.com/file.zip

# Send GET request
curl https://api.example.com/data

# Send POST request with data
curl -X POST -d "name=john" https://api.example.com/users

# Send JSON data
curl -X POST -H "Content-Type: application/json" \
     -d '{"name":"john"}' https://api.example.com/users

# Download with progress bar
curl -# -O https://example.com/file.zip

# Follow redirects
curl -L https://example.com

HTTP Methods

# GET request
curl https://api.example.com/users

# POST request
curl -X POST -d "data" https://api.example.com

# PUT request
curl -X PUT -d "data" https://api.example.com

# DELETE request
curl -X DELETE https://api.example.com/users/1

Headers and Authentication

# Add headers
curl -H "Authorization: Bearer token" https://api.example.com

# Basic authentication
curl -u username:password https://api.example.com

# Cookie usage
curl -b "session=123" https://example.com

# Save cookies
curl -c cookies.txt https://example.com

Pro Tips

  • Use -v for verbose output
  • Use -# for progress bar
  • Save commands in scripts for reuse
  • Use --compressed for compressed transfer
  • Test APIs with -i for headers

Data Handling

# Send JSON data
curl -X POST \
     -H "Content-Type: application/json" \
     -d '{"name":"john","age":30}' \
     https://api.example.com/users

# Send form data
curl -X POST \
     -F "name=john" \
     -F "photo=@image.jpg" \
     https://api.example.com/upload

# Send raw data from file
curl -X POST \
     -H "Content-Type: application/json" \
     -d @data.json \
     https://api.example.com/data

# URL-encoded data
curl -X POST \
     --data-urlencode "name=John Doe" \
     --data-urlencode "title=Software Engineer" \
     https://api.example.com/form

Advanced Options

# Follow redirects

# Set timeout
curl --connect-timeout 10 --max-time 20 https://api.example.com

# Use proxy
curl -x proxy.example.com:8080 https://api.example.com

# Ignore SSL
curl -k https://example.com

# Save cookies
curl -c cookies.txt https://example.com

# Use cookies
curl -b cookies.txt https://example.com

# Show request headers
curl -v https://example.com

# Custom DNS resolver
curl --resolve example.com:443:1.2.3.4 https://example.com

Scripting Examples

# Check HTTP status code
check_status() {
    status=$(curl -s -o /dev/null -w "%{http_code}" "$1")
    if [ $status -eq 200 ]; then
        echo "OK: $1"
    else
        echo "Error $status: $1"
    fi
}

# Monitor API endpoint
monitor_endpoint() {
    while true; do
        response=$(curl -s -w "\n%{time_total}" "$1")
        echo "$(date): $response"
        sleep 60
    done
}

# Batch API requests
batch_request() {
    for id in "$@"; do
        curl -s "https://api.example.com/users/$id" &
    done
    wait
}

Debugging Tools

# Verbose output
curl -v https://example.com

# Trace output
curl --trace output.txt https://example.com

# Format headers
curl -v https://example.com 2>&1 | grep "^>"

# Show timing
curl -w "\nTime: %{time_total}s\n" https://example.com

# Full timing details
curl -w @- https://example.com <<'EOF'
    time_namelookup:  %{time_namelookup}s\n
    time_connect:  %{time_connect}s\n
    time_appconnect:  %{time_appconnect}s\n
    time_pretransfer:  %{time_pretransfer}s\n
    time_redirect:  %{time_redirect}s\n
    time_starttransfer:  %{time_starttransfer}s\n
    time_total:  %{time_total}s\n
EOF
